# 1단계: 한글 스크립트

안녕하세요.
이것은 **테스트 문장**{test sentence}입니다.
좋은 하루 보내세요.

# 2단계: 영어 스크립트

Hello.
This is a test sentence.
Have a great day.

# 2단계: 끊어읽기 버전

Hello.
This is | a test sentence.
Have a great day.

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→영 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

안녕하세요 → Hello.
테스트 문장 → This is a test sentence.
좋은 하루 → Have a great day.

## 레벨 2: 핵심 키워드 (50% 정보)

안녕 → Hello.
테스트 → This is a test sentence.
하루 → Have a great day.

## 레벨 3: 최소한의 키워드 (30% 정보)

안녕 → Hello.
문장 → This is a test sentence.
하루 → Have a great day.

## 레벨 4: 영어 키워드

hello → Hello.
sentence → This is a test sentence.
day → Have a great day.
