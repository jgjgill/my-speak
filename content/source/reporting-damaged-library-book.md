# 1단계: 한글 스크립트

안녕하세요, 이 책을 반납하려고 하는데요.
그런데 **문제가 하나 생겼어요**{I've run into a problem}.
제가 빌린 책의 몇 페이지가 찢어져 있는 것을 발견했어요.
제가 그런 게 아닌데, **어떻게 해야 할지 모르겠어요**{I'm not sure what to do}.
혹시 제가 변상해야 하나요?
알겠습니다, 확인해 주셔서 감사합니다.

# 2단계: 외국어 스크립트

Hello, I'd like to return this book.
But I've run into a problem.
I discovered that a few pages in the book I borrowed are torn.
It wasn't me, but I'm not sure what to do.
Do I have to pay for the damage?
I see, thank you for checking.

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

책 반납, 문제 발생 → Hello, I'd like to return this book. But I've run into a problem.
페이지 찢어진 것 발견, 내가 안 그랬음 → I discovered that a few pages in the book I borrowed are torn. It wasn't me.
어떻게 해야 할지, 변상해야 하는지 → I'm not sure what to do. Do I have to pay for the damage?

## 레벨 2: 핵심 키워드 (50% 정보)

책 반납, 문제 → Hello, I'd like to return this book. But I've run into a problem.
페이지 찢어짐, 변상 여부 → A few pages are torn. Do I have to pay for the damage?

## 레벨 3: 최소한의 키워드 (30% 정보)

문제, 찢어진 책 → I have a problem with this torn book.
변상? → Do I have to pay for it?

## 레벨 4: 외국어 키워드

return book, problem → Hello, I'd like to return this book. But I've run into a problem.
pages torn, pay for damage → I discovered that a few pages are torn. Do I have to pay for the damage?
