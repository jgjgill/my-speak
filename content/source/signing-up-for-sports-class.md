# 1단계: 한글 스크립트

안녕하세요, 테니스 강좌에 대해 **문의하고 싶어서요**{I'd like to ask about}.
네, 안녕하세요! 초급반은 화요일과 목요일 저녁에 있습니다.
혹시 **자리가 남아 있나요**{are there any spots left}?
네, 아직 몇 자리 남아있어요. 바로 등록 도와드릴까요?
좋아요. 등록하려면 **무엇이 필요한가요**{what do I need to do}?
이 신청서를 작성해 주시고 신분증을 보여주시면 됩니다.

# 2단계: 외국어 스크립트

Hello, I'd like to ask about the tennis class.
Hi there! The beginner class is on Tuesday and Thursday evenings.
Are there any spots left?
Yes, there are still a few spots available. Would you like to register now?
Great. What do I need to do to sign up?
Just fill out this application form and show me your ID card, please.

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

테니스 강좌, 문의 → Hello, I'd like to ask about the tennis class.
자리 남아 있나요 → Are there any spots left?
등록, 무엇이 필요한가요 → What do I need to do to sign up?
