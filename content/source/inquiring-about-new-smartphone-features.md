# 1단계: 한글 스크립트

안녕하세요, 새로 나온 스마트폰에 대해 **물어보고 싶은데요**{I'd like to ask about}.
이 모델의 가장 큰 특징은 무엇인가요?
카메라 성능이 **향상되었다고 들었어요**{I heard the camera performance has improved}.
배터리는 한 번 충전으로 얼마나 오래 가나요?
다른 색상 옵션도 있는지 **궁금해요**{I'm curious if}.
직접 한번 만져봐도 될까요?

# 2단계: 외국어 스크립트

Hi, I'd like to ask about the new smartphone.
What is the main feature of this model?
I heard the camera performance has improved.
How long does the battery last on a single charge?
I'm curious if there are other color options.
Can I try holding it in my hand?

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

새 스마트폰, 물어보고 싶어요 → I'd like to ask about the new smartphone.
이 모델, 가장 큰 특징 → What is the main feature of this model?
카메라 성능, 향상되었다고 들었어요 → I heard the camera performance has improved.
다른 색상 옵션, 궁금해요 → I'm curious if there are other color options.

## 레벨 2: 핵심 키워드 (50% 정보)

새 스마트폰, 문의 → I'd like to ask about the new smartphone.
주요 특징 → What is the main feature of this model?
카메라 성능 향상 → I heard the camera performance has improved.
다른 색상 → I'm curious if there are other color options.

## 레벨 3: 최소한의 키워드 (30% 정보)

문의 → I'd like to ask about the new smartphone.
특징 → What is the main feature of this model?
배터리 → How long does the battery last on a single charge?

## 레벨 4: 외국어 키워드
