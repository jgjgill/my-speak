# 1단계: 한글 스크립트

안녕하세요, 이 소포를 한국으로 보내고 싶어요.
네, 이 안에 **깨지기 쉬운 물건**{any fragile items}이 있나요?
아니요, 없어요. 옷이랑 책 몇 권이에요.
알겠습니다. 배송은 일반 우편과 특급 우편 중 어떤 걸로 하시겠어요?
각각 **요금이 얼마인가요**{How much is the fee}?
일반 우편은 20달러, 특급 우편은 45달러입니다.

# 2단계: 외국어 스크립트

Hello, I'd like to send this package to Korea.
Okay, are there **any fragile items** inside?
No, there aren't. It's just some clothes and a few books.
I see. Would you like standard shipping or express shipping?
**How much is the fee** for each?
Standard is $20 and express is $45.

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

소포, 한국으로 보내기 → I'd like to send this package to Korea.
깨지기 쉬운 물건, 안에 있나요 → Are there any fragile items inside?
요금, 각각 얼마 → How much is the fee for each?

## 레벨 2: 핵심 키워드 (50% 정보)

한국으로 소포 보내기 → I'd like to send this package to Korea.
깨지기 쉬운 물건 → Are there any fragile items inside?
요금 문의 → How much is the fee for each?

## 레벨 3: 최소한의 키워드 (30% 정보)

소포 보내기 → I'd like to send this package to Korea.
내용물 확인 → Are there any fragile items inside?
요금 → How much is the fee for each?

## 레벨 4: 외국어 키워드

send package → I'd like to send this package to Korea.
fragile items → Are there any fragile items inside?
fee for each → How much is the fee for each?
