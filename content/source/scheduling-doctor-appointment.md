# 1단계: 한글 스크립트

안녕하세요, 병원입니다. 무엇을 도와드릴까요?
안녕하세요. **의사 선생님 예약**{doctor's appointment}을 하고 싶습니다.
네, **언제 방문하시겠어요?**{When would you like to come in?}
**내일 오후 3시**{tomorrow afternoon at 3 PM} 괜찮을까요?
잠시만요... 네, **그 시간 가능합니다.**{Yes, that time is available.}
**제 이름은**{My name is} **존 스미스**{John Smith}입니다.
알겠습니다. **존 스미스 씨**{Mr. John Smith}, **내일 오후 3시**{tomorrow at 3 PM}에 예약되었습니다.
감사합니다.

# 2단계: 외국어 스크립트

Hello, this is the hospital. How can I help you?
I would like to schedule a doctor's appointment.
Yes, when would you like to come in?
Would tomorrow afternoon at 3 PM be okay?
Yes, that time is available.
My name is John Smith.
Mr. John Smith, you are booked for tomorrow at 3 PM.
Thank you.
