# 1단계: 한글 스크립트

여보세요, 302호에 사는 김민준이라고 합니다.
아, 민준 씨, 안녕하세요. 무슨 일이에요?
세탁기가 작동하지 않아서 연락드렸어요.
작동을 안 한다고요? 정확히 어떤 문제인가요?
**전원이 켜지지 않아요**{it won't turn on}.
플러그가 제대로 꽂혀 있는지 확인해 보셨어요?
네, 전원 코드도 확인했는데 여전히 작동하지 않아요.
알겠습니다. **언제부터 그랬나요**{Since when has it been like that}?
어제까지는 괜찮았는데, 오늘 아침부터 안 되네요.
알겠습니다. 제가 수리 기사님을 불러서 **가능한 한 빨리 보내드릴게요**{I'll send someone over as soon as possible}.
감사합니다. 혹시 언제쯤 오실 수 있는지 아시나요?
아마 기사님이 직접 연락해서 **일정을 잡을 거예요**{schedule a visit}.

# 2단계: 외국어 스크립트

Hello, this is Minjun Kim from unit 302.
Oh, Minjun, hello. What's up?
I'm calling because the washing machine isn't working.
Not working? What's the problem exactly?
**It won't turn on** at all.
Did you check if it's plugged in properly?
Yes, I checked the power cord, but it still doesn't work.
I see. **Since when has it been like that**?
It was working fine yesterday, but it stopped working this morning.
Okay. I'll call a repairman. **I'll send someone over as soon as possible**.
Thank you. Do you know when they might come?
They will probably call you directly to **schedule a visit**.

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

302호, 세탁기 작동 안 함, 전원 안 켜짐, 오늘 아침부터, 수리 기사님 가능한 빨리 → The washing machine in my unit, 302, isn't working. It won't turn on since this morning. Could you please send a repairman as soon as possible?

## 레벨 2: 핵심 키워드 (50% 정보)

세탁기 고장, 전원 문제, 수리 요청 → My washing machine is broken. It seems to be a power issue. I'd like to request a repair.

## 레벨 3: 최소한의 키워드 (30% 정보)

세탁기 수리 필요 → The washing machine needs to be repaired.

## 레벨 4: 외국어 키워드

Washing machine, not working, send repairman → The washing machine isn't working. Can you send a repairman?
