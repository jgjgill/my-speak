# 1단계: 한글 스크립트

안녕하세요, 도서관 회원증을 만들고 싶어요.
네, 물론입니다. 처음 만드시는 건가요?
네, 처음이에요. 회원증을 만들려면 **무엇이 필요한가요**{what do I need}?
신분증과 주소를 증명할 수 있는 서류가 필요합니다.
여기 제 신분증과 공과금 고지서가 있어요.
감사합니다. 이 **신청서를 작성해 주세요**{fill out this application form}.
네, 다 작성했어요. 여기 있습니다.
좋습니다. 잠시만 기다리시면 회원증이 발급될 거예요.

# 2단계: 외국어 스크립트

Hello, I'd like to make a library card.
Yes, of course. Is this your first time making one?
Yes, it's my first time. What do I need to make a card?
You'll need an ID and a document that can prove your address.
Here is my ID and a utility bill.
Thank you. Please fill out this application form.
Okay, I've filled it all out. Here you go.
Great. If you wait just a moment, your card will be issued.

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

도서관 회원증, 만들고 싶어요 → I'd like to make a library card.
필요한 것, 신분증, 주소 증명 서류 → You'll need an ID and a document that can prove your address.
신청서, 작성 완료 → I've filled out the application form.

## 레벨 2: 핵심 키워드 (50% 정보)

회원증, 만들기 → I'd like to make a library card.
필요한 서류 → What documents do I need?
신청서 작성 → Please fill out this application form.

## 레벨 3: 최소한의 키워드 (30% 정보)

회원증 신청 → I'd like to apply for a card.
필요한 것? → What do I need?
다 했어요 → I'm done.

## 레벨 4: 외국어 키워드

Library card, make → I'd like to make a library card.
Need, documents → What documents do I need?
Fill out, form → Please fill out this form.
