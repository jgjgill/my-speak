# 1단계: 한글 스크립트

이 새 보드게임 정말 기대된다!
먼저, **게임판을 테이블 중앙에 놓자**{let's put the game board in the center of the table}.
각자 원하는 색깔의 말을 고르면 돼.
내가 규칙을 간단하게 설명해 줄게.
우리의 목표는 **가장 먼저 결승점에 도착하는 거야**{to be the first to reach the finish line}.
질문 있으면 언제든지 물어봐.

# 2단계: 외국어 스크립트

I'm so excited about this new board game!
First, let's put the game board in the center of the table.
We can each choose a piece of the color we want.
I'll briefly explain the rules to you.
Our goal is to be the first to reach the finish line.
If you have any questions, feel free to ask anytime.

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

새 보드게임, 기대 / 게임판, 중앙에 놓기 → I'm so excited about this new board game! First, let's put the game board in the center of the table.
규칙 설명, 목표, 결승점 도착 → I'll briefly explain the rules. Our goal is to be the first to reach the finish line.

## 레벨 2: 핵심 키워드 (50% 정보)

원하는 색깔, 선택 / 질문 있으면 물어봐 → We can each choose a piece of the color we want. If you have any questions, feel free to ask.
목표, 결승점 → Our goal is to be the first to reach the finish line.

## 레벨 3: 최소한의 키워드 (30% 정보)

규칙 설명 → I'll briefly explain the rules to you.
질문 → If you have any questions, feel free to ask anytime.

## 레벨 4: 외국어 키워드

new board game → I'm so excited about this new board game!
rules, goal → I'll briefly explain the rules. Our goal is to be the first to reach the finish line.
