# 1단계: 한글 스크립트

다음 주에 프로젝트 현황을 발표해야 해요.
네, **어떤 내용을 포함해야 할까요?**{What should we include?}
**현재까지 진행된 작업**{work done so far}과 **앞으로 할 일**{tasks ahead}을 중심으로 설명해야 해요.
**발표 시간은 얼마나 되나요?**{How much time do we have for the presentation?}
10분 정도입니다. **간단하고 명확하게**{simply and clearly} 전달하는 것이 중요해요.
알겠습니다. **슬라이드를 준비하는 것을 도울게요.**{I can help prepare the slides.}

# 2단계: 외국어 스크립트

I need to present the project status next week.
Yes, what should we include?
We should explain the work done so far and the tasks ahead.
How much time do we have for the presentation?
About 10 minutes. It's important to convey it simply and clearly.
Okay. I can help prepare the slides.

# 3단계: 사용자 자유 연습

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

다음 주, 프로젝트 현황, 발표해야 해요 → I need to present the project status next week.
네, 어떤 내용, 포함해야 할까요? → Yes, what should we include?
현재까지 진행된 작업, 앞으로 할 일, 중심으로, 설명해야 해요 → We should explain the work done so far and the tasks ahead.
발표 시간, 얼마나 되나요? → How much time do we have for the presentation?
10분, 간단하고 명확하게, 전달, 중요해요 → About 10 minutes. It's important to convey it simply and clearly.
알겠습니다, 슬라이드, 준비하는 것, 도울게요 → Okay. I can help prepare the slides.
