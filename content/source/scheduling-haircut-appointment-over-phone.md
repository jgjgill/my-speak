# 1단계: 한글 스크립트

안녕하세요, **헤어컷 예약하고 싶은데요**{book a haircut}.
이번 주 토요일에 **예약 가능한가요**{have any openings}?
오전 11시가 좋을 것 같아요.
네, 제 이름은 김민수예요.
감사합니다, **그때 뵙겠습니다**{see you then}!

# 2단계: 외국어 스크립트

Hello, I'd like to book a haircut.
Do you have any openings this Saturday?
11 AM would be great.
Yes, my name is Minsu Kim.
Thank you, see you then!

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

이번 주 토요일, 예약 가능? → Do you have any openings this Saturday?
오전 11시 → 11 AM would be great.

토요일 가능? → Do you have any openings this Saturday?
11시 → 11 AM would be great.
