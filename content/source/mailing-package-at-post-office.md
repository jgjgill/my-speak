# 1단계: 한글 스크립트

이 소포를 한국으로 보내고 싶어요.
안에 뭐가 들어있나요?
옷이랑 책 몇 권이요. **깨지기 쉬운 물건은**{fragile items} 없어요.
알겠습니다. 일반 우편이랑 특급 우편 중에 **어떤 걸로 하시겠어요**{which one would you like}?
특급 우편은 **얼마나 걸리나요**{how long does it take}?
보통 3일에서 5일 정도 걸려요.

# 2단계: 외국어 스크립트

I'd like to send this package to Korea.
What's inside?
Some clothes and a few books. There are no **fragile items**.
Okay. Standard or express mail, **which one would you like**?
**How long does it take** for express mail?
It usually takes about 3 to 5 days.

# 2단계: 끊어읽기 버전

I'd like to send | this package | to Korea.
What's inside?
Some clothes | and a few books. || There are no | fragile items.
Okay. || Standard or express mail, | which one | would you like?
How long | does it take | for express mail?
It usually takes | about 3 to 5 days.

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

소포, 한국으로 보내기 → I'd like to send this package to Korea.
깨지기 쉬운 물건, 없음 → There are no fragile items.
특급 우편, 얼마나 걸리는지 → How long does it take for express mail?

## 레벨 2: 핵심 키워드 (50% 정보)

한국으로 보내기 → I'd like to send this package to Korea.
특급 우편, 소요 시간 → How long does it take for express mail?

## 레벨 3: 최소한의 키워드 (30% 정보)

소포 보내기 → I'd like to send this package to Korea.
소요 시간 → How long does it take?

## 레벨 4: 외국어 키워드

send package → I'd like to send this package to Korea.
how long → How long does it take?
