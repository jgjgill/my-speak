# 1단계: 한글 스크립트

레시피를 다시 한번 확인해볼까?
응, 우리가 **재료를 다 넣었는지**{if we've added all the ingredients} 확실히 해야 해.
설탕이랑 버터는 넣었고, 이제 밀가루를 **체에 쳐서 넣을**{sift in} 차례야.
반죽이 너무 질어지지 않게 **조심해야겠다**{I should be careful}.
오븐 예열은 다 됐어?
응, 이제 오븐에 넣고 15분만 기다리면 돼. **냄새가 정말 좋다**{It smells so good}!

# 2단계: 외국어 스크립트

Shall we check the recipe one more time?
Yes, we need to make sure if we've added all the ingredients.
We put in the sugar and butter, and now it's time to sift in the flour.
I should be careful so the dough doesn't get too sticky.
Is the oven preheated?
Yes, now we just need to put it in the oven and wait for 15 minutes. It smells so good!

# 3단계: 사용자 자유 연습

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

레시피 확인, 모든 재료, 밀가루 체 치기 → Shall we check the recipe to make sure if we've added all the ingredients? Now it's time to sift in the flour.

## 레벨 2: 핵심 키워드 (50% 정보)

반죽 조심, 오븐 예열 → I should be careful with the dough. Is the oven preheated?

## 레벨 3: 최소한의 키워드 (30% 정보)

15분, 좋은 냄새 → We just need to wait for 15 minutes. It smells so good!

## 레벨 4: 외국어 키워드

check recipe, all ingredients, preheated → Shall we check the recipe one more time? Have we added all the ingredients? Is the oven preheated?
