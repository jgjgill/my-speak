# 1단계: 한글 스크립트

[5-8문장의 자연스러운 한글 대화]
[학습 포인트는 **한글표현**{외국어 번역} 형식으로 표시]

A: 죄송해요, 제가 늦었어요.
B: 어서 와요! **괜찮아요**{It's okay}.
A: 오늘 아침에 **교통 체증이 심했어요**{The traffic was terrible this morning}.
B: 이해해요. 그럴 수 있죠.
A: **다음에는 제시간에 오도록 노력할게요**{I'll try to be on time next time}.
B: 그렇게 말해줘서 좋네요.
A: 기다려주셔서 감사합니다.

# 2단계: 외국어 스크립트

A: I'm so sorry I'm late.
B: Oh, hi! It's okay.
A: The traffic was terrible this morning.
B: I understand. It happens.
A: I'll try to be on time next time.
B: That's good to hear.
A: Thank you for waiting.
