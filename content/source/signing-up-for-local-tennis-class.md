# 1단계: 한글 스크립트

안녕하세요, 테니스 강좌에 대해 문의하고 싶어요.
네, 물론이죠. 초급반과 중급반이 있는데, 어떤 걸 찾으세요?
저는 **완전 초보예요**{I'm a complete beginner}. 초급반은 언제 수업이 있나요?
초급반은 매주 화요일과 목요일 저녁 7시에 있습니다.
좋아요. **수강 신청하려면**{To sign up for the class} 어떻게 해야 하나요?
이 신청서를 작성해주시고, 수강료를 결제하시면 됩니다.

# 2단계: 외국어 스크립트

Hello, I'd like to inquire about the tennis classes.
Yes, of course. We have beginner and intermediate classes. Which one are you looking for?
I'm a complete beginner. When are the beginner classes?
The beginner class is every Tuesday and Thursday at 7 PM.
Great. How do I sign up for the class?
Please fill out this application form and pay the course fee.

# 3단계: 사용자 자유 연습

1 ~ 2단계에서 구성한 콘텐츠 기반으로 한→외국어 스피킹 연습 진행

# 4단계: 키워드 스피치

## 레벨 1: 상세한 키워드 (70% 정보)

테니스 강좌 문의, 완전 초보, 수업 시간, 수강 신청 방법 → Hello, I'd like to inquire about the tennis classes. I'm a complete beginner and would like to know the class schedule and how to sign up.

## 레벨 2: 핵심 키워드 (50% 정보)

테니스 문의, 초급반, 신청 방법 → I'd like to ask about the beginner tennis class and how to register.

## 레벨 3: 최소한의 키워드 (30% 정보)

테니스 강좌 등록 → I want to register for a tennis class.

## 레벨 4: 외국어 키워드

Inquire, Beginner tennis, Sign up → I'd like to inquire about the beginner tennis class and sign up.
